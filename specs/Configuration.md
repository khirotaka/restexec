# 環境変数

| 変数名 | デフォルト値 | 説明 |
|-------|-----------|------|
| `PORT` | 3000 | HTTPサーバーのリスニングポート |
| `WORKSPACE_DIR` | /workspace | コード実行ディレクトリ |
| `TOOLS_DIR` | /tools | 共有ツールディレクトリ |
| `DEFAULT_TIMEOUT` | 5000 | デフォルトタイムアウト（ミリ秒） |
| `MAX_TIMEOUT` | 300000 | 最大タイムアウト（ミリ秒） |
| `LOG_LEVEL` | info | ログレベル (debug, info, warn, error) |
